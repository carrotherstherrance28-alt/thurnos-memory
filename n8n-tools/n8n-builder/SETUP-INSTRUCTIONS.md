# AI Newsletter Workflow - Setup Instructions

## Overview
This workflow automatically generates and sends a daily AI automation newsletter at 7:00 AM Central Time using:
- **Perplexity API** for research
- **OpenRouter (Claude 3.5 Sonnet)** for writing
- **Gmail** for delivery

---

## Import Instructions

1. **Open n8n** and navigate to your workflows
2. Click **"Import from File"** or **"Import from URL"**
3. Select the `ai-newsletter-workflow.json` file
4. The workflow will be imported with all nodes and connections

---

## Required Credentials Setup

### 1. Perplexity API Key (HTTP Header Auth)

**Create credential:**
- **Credential Type:** Header Auth
- **Credential Name:** `Perplexity API Key`
- **Configuration:**
  - **Header Name:** `Authorization`
  - **Header Value:** `Bearer YOUR_PERPLEXITY_API_KEY`

**Get your API key:**
- Visit: https://www.perplexity.ai/settings/api
- Create an account and generate an API key
- Copy the key and paste it in the format: `Bearer pplx-...`

---

### 2. OpenRouter API Key (HTTP Header Auth)

**Create credential:**
- **Credential Type:** Header Auth
- **Credential Name:** `OpenRouter API Key`
- **Configuration:**
  - **Header Name:** `Authorization`
  - **Header Value:** `Bearer YOUR_OPENROUTER_API_KEY`

**Get your API key:**
- Visit: https://openrouter.ai/keys
- Create an account and generate an API key
- Copy the key and paste it in the format: `Bearer sk-or-v1-...`
- Add credits to your account for API usage

---

### 3. Gmail OAuth2 Credential

**Create credential:**
- **Credential Type:** Gmail OAuth2 API
- **Credential Name:** `Gmail Account`
- **Configuration:**
  - Follow n8n's built-in Gmail OAuth2 setup wizard
  - Grant permissions to send emails
  - Use your Google account: `ctherrance@gmail.com`

**Setup steps:**
1. In n8n, create a new Gmail OAuth2 credential
2. Click "Connect my account"
3. Sign in with your Google account
4. Grant the requested permissions
5. Save the credential

## Workflow Configuration

### Schedule Trigger Settings
- **Cron Expression:** `0 7 * * *` (7:00 AM daily)
- **Timezone:** `America/Chicago` (Central Time)
- The workflow is set to trigger automatically every morning

### Perplexity Research Agent
- **Model:** `sonar` (Perplexity's research model)
- **Search Recency:** Last 24-48 hours
- **Focus Areas:**
  - AI automation tools and platforms
  - Voice agent technologies
  - Agentic workflow innovations
  - API integrations and updates
  - Practical business use cases

### OpenRouter Claude Writer
- **Model:** `anthropic/claude-3.5-sonnet`
- **Output:** Professional HTML newsletter with:
  - Catchy subject line
  - Engaging introduction
  - Well-organized content sections
  - Actionable insights
  - Mobile-responsive design
  - Inline CSS styling

### Gmail Delivery
- **Recipient:** `ctherrance@gmail.com`
- **Format:** HTML email
- **Subject:** Generated by Claude or fallback with date

## Troubleshooting

### Common Issues

**1. Perplexity API Errors**
- Check API key format: `Bearer pplx-...`
- Verify account has credits/active subscription
- Check API rate limits

**2. OpenRouter API Errors**
- Verify API key format: `Bearer sk-or-v1-...`
- Ensure account has sufficient credits
- Check model availability: `anthropic/claude-3.5-sonnet`

**3. Gmail Sending Errors**
- Re-authenticate Gmail OAuth2 credential
- Check Google account permissions
- Verify email address is correct

**4. JSON Parsing Errors**
- The "Extract Subject & HTML" node uses regex to parse Claude's response
- If format changes, you may need to adjust the extraction logic
- Check Claude's actual output format in the node execution data

**5. Timezone Issues**
- Verify workflow settings show `America/Chicago`
- Check your n8n instance timezone settings
- Cron expression `0 7 * * *` = 7:00 AM in configured timezone

## Customization Options

### Change Newsletter Frequency
Edit the Schedule Trigger cron expression:
- **Twice daily:** `0 7,19 * * *` (7 AM and 7 PM)
- **Weekdays only:** `0 7 * * 1-5` (Monday-Friday)
- **Weekly:** `0 7 * * 1` (Every Monday)

### Add More Recipients
Modify the Gmail node:
- Change `toEmail` to include multiple addresses (comma-separated)
- Or add a loop to send to multiple recipients

### Customize Research Topics
Edit the Perplexity API request body:
- Modify the system prompt
- Adjust the user prompt focus areas
- Change `search_recency_filter` (day, week, month)

### Adjust Newsletter Style
Edit the OpenRouter request:
- Modify the system prompt for different tone/style
- Adjust temperature for creativity (0.5-1.0)
- Request different HTML layouts
